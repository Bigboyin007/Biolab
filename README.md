# BioChar Lab - Colorful Edition

A vibrant, modern web application for environmental engineering research data analysis.

## Features
- **XRD Analysis** with peak detection and d-spacing calculation
- **FTIR Analysis** with functional group identification
- **BET Analysis** with surface area calculation and isotherm classification
- **Dark theme** with colorful gradients and animations
- **Fully responsive** design

## Deploy to GitHub Pages
1. Create new repo on GitHub
2. Upload all files maintaining folder structure
3. Settings → Pages → Deploy from branch → main → Save
4. Visit `https://yourusername.github.io/biochar-lab/`
